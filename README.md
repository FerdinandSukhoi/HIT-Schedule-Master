## 主要功能

- 从校园网中下载个人课表或者班级推荐课表并将其转换为 iCalendar (RFC 5545) 格式以便导入到日历软件中
- 自动检查更新

## 一些特性

- 所有通过“个人课表”生成的日历，默认在课程开始前25分钟进行提醒
- 通过“班级推荐课表”生成的日历，默认不进行提醒
